QuickLaunch Elements

Quick Launch is a menu of links that can appear on the left side of each page in a website. When it is displayed, it can help users navigate to important content.

Quick Launch Navigation Menu is a menu that allows users to access content that is specific to the site they are on via bookmarks (URLs)
