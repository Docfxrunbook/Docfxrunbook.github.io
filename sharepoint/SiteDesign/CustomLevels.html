Custom Permission Level (Roles)

Default permission levels are made up of a set of permissions that enable users to perform a collection of related tasks. SharePoint includes seven permission levels. You can customize the permissions contained within most of these permission levels.

Some of the permission levels include Contribute, Edit, Design, Read Only, etc.

Customized permission levels can be created to include/exclude these permissions (add item, delete item, approve items, view items etc.)
