Dynamic Groups

Dynamic configuration of security group membership for Azure Active Directory (Azure AD) is available in the Azure portal. Azure AD Premium P1 or P2 licensing is required to create and use dynamic groups.

Members can be automatically added to or removed from a security group based on their attributes. These groups can provide access to applications or cloud resources (SharePoint sites, documents) and to assign licenses to members

Administrators can set rules to populate groups that are created in Azure AD based on user attributes (such as userType, department, or country/region).
